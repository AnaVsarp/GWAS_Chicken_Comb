# Lövsta comb data

Contents:
* `Chicken_50K_CobbCons_15000986_A.csv` describes the design of the genotyping array used.
* `genotypes.{map, ped}` contains the data genotype data in Plink format
* `genotypes.raw` contains the genotypes coded 0,1,2
* `comb_g.csv` contains bird and comb masses in grams
* `pheno.csv` contains other phenotypes and covariate
* `ids_to_filter` is a folder with three text files that indicate birds that should be filtered out of the genotype data for quality control reasons
